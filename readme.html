# TechnoShop ![TechnoShop Logo](wp-content/uploads/2024/12/Technoshop_logo.jpg) Bienvenue sur **TechnoShop**, votre boutique en ligne pour les produits high-tech de qualité. [Contact](#contact) --- ## Aperçu du Projet **TechnoShop** est une plateforme e-commerce spécialisée dans la vente de produits high-tech, conçue pour offrir : - Une navigation fluide. - Une gestion efficace des produits. - Un design réactif et adapté à tous les types d'appareils. ### Pages Principales - **Accueil** : Vitrine dynamique mettant en avant les promotions et produits phares. - **Boutique** : Catalogue complet avec filtres avancés. - **Mon Compte** : Gestion des profils et commandes. - **Contact** : Formulaire intuitif pour les questions et demandes. --- ## Fonctionnalités - **Interface Responsive** : Compatible avec smartphones, tablettes, et ordinateurs. - **Recherche Avancée** : Tri par prix, marque, et catégorie. - **Paiements Sécurisés** : Support de PayPal et Stripe. - **Gestion des Produits** : Catégories claires, fiches produits détaillées. - **Avis Clients** : Encouragement des retours d’expérience. --- ## Technologies Utilisées - **WordPress** : Gestion de contenu principale. - **WooCommerce** : Plugin pour la gestion e-commerce. - **Thème Big Store** : Design optimisé pour les boutiques en ligne. - **Extensions Utiles** : - **Elementor** : Personnalisation facile des pages. - **Smush** : Optimisation des images. - **WooCommerce Currency Switcher** : Support multi-devises. --- ## Installation 1. Clonez le dépôt : ```bash git clone https://github.com/ZakariaHanani/TechnoShop_wordpress.git ``` 2. Installez un environnement local (XAMPP ou MAMP). 3. Configurez WordPress avec la base de données MySQL (nommée `technoshop_db`). 4. Importez les fichiers dans le dossier `htdocs` de votre serveur local. 5. Accédez au site via `localhost/technoshop`. --- ## Collaboration Le projet a été réalisé par : - **Moumen Mariam** - **Hanani Zakaria** - **Tagarte Abdessadek** ### Outils de Collaboration - **GitHub** : Gestion des versions et des contributions. - **Branches** : Séparation du travail pour éviter les conflits. - **Pull Requests** : Validation des modifications avant intégration. --- ## Objectifs 1.
